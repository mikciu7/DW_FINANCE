# Datawarehouse w finansach

To jest projekt majacy na celu:
 - zczytanie danych finansowych
 - przetworzenie ich
 - wystawienie do analizy
 - dodatkowo mozna stworzyc modele ml korzystajace z tych danych


## Struktura Projektu
 - INGEST - Pobranie danych
 - DATA_TRANSFORM - Przeksztalcenie danych
